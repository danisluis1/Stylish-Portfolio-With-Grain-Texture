##Place Image Into Photoshop

    - Place Background Image
	- Move Picture between Layer1 And Gradient Layout.
	+ Create Area Selection from Rectangle 1[Click Mouse Left]
	+ Create Picture => Layer Mask => Ctrl + I(Like Circle)
	+ Duplicate this Layer => Move onside Rectangle 1. Ctrl + I
	+ Create Gradient and Create Animation Gradient from Image On Top
	+ Create Layer Curvor On Layer Image [0/12 - 255/277]
	+ Create Layer Brighness: -16/-13
